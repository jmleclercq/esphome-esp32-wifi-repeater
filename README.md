# ESP32 WiFi Repeater / Router (ESPHome)

![ESPHome](https://img.shields.io/badge/ESPHome-Compatible-3DDC84)
![ESP32](https://img.shields.io/badge/Platform-ESP32-blue)
![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

A lightweight ESP32 WiFi repeater configuration using ESPHome.

## Features
- WiFi repeater mode
- OTA updates
- Simple configuration
- Optional Home Assistant integration

## Quick Start
1. Install ESPHome
2. Copy config/example-router.yaml
3. Configure secrets.yaml
4. Flash with `esphome run your-router.yaml`
